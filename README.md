# App Support

Welcome to the support page for our app.

This app accesses the photo library only to let users select images for editing.  
The app does **not** collect, store, or share any photos.  
Network access is used only to validate a one-time activation code on first launch.  
No other user data is collected or shared.

## How to Get Help

If you encounter issues with the app or need guidance on using its features, you can contact us:

- **Email:** workwplus@gmail.com

### Notes

- This app only edits images.  
- It does not print, connect to any printers, or interact with any external hardware.
